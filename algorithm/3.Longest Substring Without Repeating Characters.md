# [3. Longest Substring Without Repeating Characters](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

## Description

Given a string `s`, find the length of the **longest** **substring** without duplicate characters.

**Example 1:**

```
Input: s = "abcabcbb"
Output: 3
Explanation: The answer is "abc", with the length of 3. Note that "bca" and "cab" are also correct answers.
```

**Example 2:**

```
Input: s = "bbbbb"
Output: 1
Explanation: The answer is "b", with the length of 1.
```

**Example 3:**

```
Input: s = "pwwkew"
Output: 3
Explanation: The answer is "wke", with the length of 3.
Notice that the answer must be a substring, "pwke" is a subsequence and not a substring.
```

 

**Constraints:**

- `0 <= s.length <= 5 * 104`
- `s` consists of English letters, digits, symbols and spaces.



## Solution 1

自解。

**思路**：使用一个动态滑动窗口来减少遍历的次数，**首先保证窗口内没有重复字符**，然后控制滑动窗口，判断窗口长度与历史最大无重复字符子串的长度，并更新大无重复字符子串长度最大值。

**窗口控制**主要分为两步如下：

* **窗口扩充**：检查窗口尾端的下一个元素在窗口中无相同字符，更新窗口尾端，窗口长度。
* **窗口滑动**：当窗口尾端的下一个元素在窗口中有相同字符，更新**窗口头位置**为**窗口内相同字符的洗一个元素**位置，**窗口尾端**更新为**窗口尾端的下一个元素**。

原理：由于不用记录子串位置，仅需要记录最大长度，所以仅需要保留最大值，窗口扩充原理不用说明，窗口变更的原理为，窗口内首先不存在相同字符串，而窗口尾端的下一个元素。与窗口内有且仅有一个字符相同，也就是说这两个相同字符中间所有的字符都不相同（包括这两个字符）。所以中间这段字符串就无需再做重复检查，直接从

 [**窗口内相同字符的洗一个元素**, **窗口尾端的下一个元素**]

的窗口基础上继续就可以了。

具体实现：

~~~c
int lengthOfLongestSubstring(char* s) {
    int substr_len = 0;
    int max_len = 0;
    bool iFlag = false;
    int substr_head, substr_tail, substr_cur, res_cur;

    if (NULL == s)
        return -1;

    if (strlen(s) <= 1)
        return strlen(s);
	/* 初始化 下标 长度 信息 */
    substr_head = substr_tail = substr_cur = 0;
    res_cur = substr_tail + 1;
    max_len = substr_len = 1;
    
    while(s[res_cur] != '\0') {
        iFlag = true; //窗口变更标识 false 时说明窗口发生变更
        for (substr_cur = substr_head; substr_cur <= substr_tail; substr_cur++) {
            if (s[res_cur] == s[substr_cur])
            { // 发现相同字符开始变更窗口
                substr_head = substr_cur + 1;
                substr_tail = res_cur;
                res_cur = res_cur + 1;
                substr_len = substr_tail - substr_head + 1;
                iFlag = false;
                break;
            }
        }
        if (iFlag) { //窗口未发生变更 正常扩充
            substr_len++;
            substr_tail = res_cur;
            res_cur = res_cur + 1;
        }
        /* 更新最大长度 */
        max_len = max_len > substr_len ? max_len : substr_len;
    }
    return max_len;
}
~~~
